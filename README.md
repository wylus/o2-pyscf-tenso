# O₂ Quantum Coherence with PySCF and TENSO

A reproducible workflow for studying electronic coherence in molecular oxygen using multireference electronic structure calculations (PySCF) and open quantum system dynamics (TENSO).

## Features

- ROHF electronic structure
- CAS(2,2)
- CAS(6,4)
- Bond-length scan
- Vibronic parameter extraction
- HEOM simulations with TENSO
- Temperature study
- Vibronic coupling study
- Bath-width study
